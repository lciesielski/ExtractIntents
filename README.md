# Extract Intents From Predefined JSON Files

## About The Project

Simple Application to automate some potential manual work of extracting intents for Einstein machine learning

## Getting Started

You will need to clone this repo, compile solution (it was build using `Visual Studio 19 Community`), also you will need to have predefined JSON files consistent with implemented structure.

### Prerequisites

1. Visual Studio 2019
2. Predefined JSON Files

### Installation

1. Clone this repository by running command 
   ```sh
   git clone https://github.com/lciesielski/ExtractIntents.git
   ```
2. In folder containing exe file you need to create folder named `intents` and place all JSON files inside
3. In folder containing exe file you need to create folder named `csvIntents` and leave it empty.

<!-- USAGE EXAMPLES -->
## Usage

Simply run `ExtractIntents.exe`
Results will be placed in `csvIntents`
